# Atalhos de Teclado

Atalhos disponíveis.

## Principais

| Atalho | Ação |
|--------|------|
| `F1` | Alternar entre Running e Edit |
| `F2` | Mostrar/ocultar FPS counter |
| `ESC` | Fechar overlay |

## Modo Edit

No modo Edit:
- **Clique e arraste**: Move widget
- **F1**: Volta ao modo Running

## Modo Running

No modo Running:
- **F1**: Entra no modo Edit
- **ESC**: Fecha overlay

Veja [Interface](interface.md).
