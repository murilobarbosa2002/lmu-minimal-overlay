# Interface

Modos de operação do overlay.

## Running Mode

Overlay transparente para corridas.

- Transparente
- Click-through
- Apenas visualização

**Atalho**: Padrão ao iniciar

## Edit Mode

Posicione widgets.

- Fundo visível
- Captura mouse
- Drag & drop

**Atalho**: `F1`

## Alternando Modos

Pressione `F1` para alternar entre Running e Edit.

Veja [Quick Start](../../getting-started/quick-start.md).
