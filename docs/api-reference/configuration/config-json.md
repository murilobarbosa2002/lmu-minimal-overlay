# config.json

Configurações globais da aplicação.

## Localização

```
config/config.json
```

## Estrutura

```json
{
  "colors": {},
  "thresholds": {},
  "window": {},
  "performance": {},
  "telemetry": {}
}
```

## Campos

### colors

Paleta RGB (0-255).

### thresholds

Limites para indicadores.

### window

Dimensões e transparência.

### performance

FPS target.

### telemetry

Provider e configurações.

Veja [Configuration](../../getting-started/configuration.md) para detalhes completos.
