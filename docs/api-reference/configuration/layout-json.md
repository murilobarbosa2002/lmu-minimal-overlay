# layout.json

Posições e visibilidade dos widgets.

## Localização

```
config/layout.json
```

## Estrutura

```json
{
  "widgets": [
    {
      "id": "speedometer",
      "type": "Speedometer",
      "x": 100,
      "y": 100,
      "width": 200,
      "height": 100,
      "visible": true
    }
  ]
}
```

## Campos

- `id`: Identificador único
- `type`: Classe do widget
- `x`, `y`: Posição em pixels
- `width`, `height`: Dimensões
- `visible`: Visibilidade

Veja [Configuration](../../getting-started/configuration.md).
